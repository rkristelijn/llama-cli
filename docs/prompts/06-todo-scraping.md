# Prompt 06: Automate TODO scraping to TECHDEBT.md

## Context

- `scripts/dev/todo.sh` already scans for TODOs in markdown and source code
- `make todo` runs it but only displays output to terminal
- We need it to also write to `TECHDEBT.md` for CMMI 1 check 1.3
- The script scans: markdown `- [ ]` items and `TODO|FIXME|HACK|XXX` in `src/` and `include/`

## Task

Update `scripts/dev/todo.sh` to also generate `TECHDEBT.md` at the project root.

## Change: Update `scripts/dev/todo.sh`

Find this exact text at the end of the file:

```bash
main() {
  echo "==> Markdown TODOs"
```

Replace the entire `main()` function with:

```bash
main() {
  echo "==> Markdown TODOs"
  find . -name "*.md" -not -path "./build/*" -not -path "./.git/*" -exec awk '
    FNR == 1 {
      if (prev_line != "") printf "%s:%d:%s\n", prev_file, prev_lnum, prev_line;
      prev_line = "";
    }
    /- \[ \]/ {
      match($0, /[^ ]/);
      curr_indent = RSTART;
      if (prev_line != "") {
        if (curr_indent > prev_indent) {
          printf "%s:%d:\033[1m%s\033[0m\n", FILENAME, prev_lnum, prev_line;
        } else {
          printf "%s:%d:%s\n", FILENAME, prev_lnum, prev_line;
        }
      }
      prev_line = $0;
      prev_indent = curr_indent;
      prev_lnum = FNR;
      prev_file = FILENAME;
    }
    END {
      if (prev_line != "") printf "%s:%d:%s\n", FILENAME, prev_lnum, prev_line;
    }' {} +

  echo ""
  echo "==> Code TODOs"
  grep -rn "TODO\|FIXME\|HACK\|XXX" src/ include/ --include="*.cpp" --include="*.h" 2>/dev/null || true

  # Generate TECHDEBT.md
  # @see docs/adr/adr-048-quality-framework.md §3.3 check 1.3
  {
    echo "# Technical Debt Log"
    echo ""
    echo "Auto-generated by \`make todo\` on $(date -u +%Y-%m-%d)."
    echo ""
    echo "## Code TODOs"
    echo ""
    local code_count=0
    while IFS= read -r line; do
      echo "- \`${line}\`"
      (( code_count++ )) || true
    done < <(grep -rn "TODO\|FIXME\|HACK\|XXX" src/ include/ --include="*.cpp" --include="*.h" 2>/dev/null || true)
    echo ""
    echo "**Total code TODOs: ${code_count}**"
    echo ""
    echo "## Markdown TODOs (unchecked items)"
    echo ""
    local md_count=0
    while IFS= read -r line; do
      echo "- \`${line}\`"
      (( md_count++ )) || true
    done < <(grep -rn "\- \[ \]" . --include="*.md" --exclude-dir=build --exclude-dir=.git --exclude="TECHDEBT.md" 2>/dev/null || true)
    echo ""
    echo "**Total markdown TODOs: ${md_count}**"
  } > TECHDEBT.md

  echo ""
  echo "==> TECHDEBT.md updated"
}
```

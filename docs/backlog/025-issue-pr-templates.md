# 025: Issue and PR Templates

*Status*: Idea · *Date*: 2026-04-19 · *CMMI*: 0.8 · *Issue*: [#89](https://github.com/rkristelijn/llama-cli/issues/89)

## Problem

No `.github/ISSUE_TEMPLATE/` or `PULL_REQUEST_TEMPLATE.md`. Tasks lack value statements and acceptance criteria.

## Idea

- Feature template: value statement + AC in Given-When-Then
- Bug template: reproduction steps + expected vs actual
- PR template: checklist (tests, docs, CMMI checks)

### Stacking effect

- Definition of Ready baseline
- Sprint planning input
- Review consistency

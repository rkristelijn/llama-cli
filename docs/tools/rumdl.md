# rumdl

| | |
|---|---|
| Version | 0.1.73 (pinned in `.config/versions.env`) |
| Purpose | Markdown linter written in Rust |
| Config | `.config/rumdl.toml` |
| Make target | `make markdownlint` |
| Official docs | <https://github.com/swlkr/rumdl> |

## Usage

```bash
make markdownlint
```

## Notes

- Fast Rust-based alternative to traditional Node.js markdown linters
- Config in `.config/rumdl.toml` defines enabled rules and exceptions
